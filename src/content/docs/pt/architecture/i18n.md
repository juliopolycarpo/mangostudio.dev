## Como Funciona

### Sistema De Tipos

`types.ts` define:

```typescript
type DeepStringRecord<T> = {
  [K in keyof T]: T[K] extends string
    ? string
    : T[K] extends object
      ? DeepStringRecord<T[K]>
      : never;
};

type Messages = DeepStringRecord<typeof ptBR>;
type Locale = 'pt-BR' | 'en';
```

Isso garante que `en.ts` tenha todas as mesmas chaves de `pt-BR.ts`, embora os valores possam ser strings diferentes.

### Hook React

`useI18n()` retorna `{ t, locale, setLocale }`:

```tsx

```
